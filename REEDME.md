 DecodeBot - Rule-Based AI Chatbot

 Project Overview

DecodeBot is a simple Rule-Based AI Chatbot developed using Python as part of the **Artificial Intelligence Internship Program at DecodeLabs**.

The chatbot interacts with users through predefined responses using **if-else statements**, loops, and basic decision-making logic. This project demonstrates fundamental AI concepts without using machine learning.

 Features

* Greeting responses
* Answer common questions
* Display current date and time
* Perform basic arithmetic operations
* Tell jokes
* Show help menu
* Handle unknown inputs
* Exit the chatbot gracefully

 Technologies Used

* Python 3
* VS Code
* Control Flow
* If-Else Statements
* While Loop
* Datetime Module

 Available Commands

| User Input        | Bot Response                 |
| ----------------- | ---------------------------- |
| hello             | Greets the user              |
| hi                | Greets the user              |
| how are you       | Responds with its status     |
| what is your name | Introduces itself            |
| who created you   | Displays creator information |
| date              | Shows current date           |
| time              | Shows current time           |
| tell me a joke    | Tells a programming joke     |
| add               | Performs addition            |
| subtract          | Performs subtraction         |
| multiply          | Performs multiplication      |
| divide            | Performs division            |
| help              | Displays all commands        |
| bye               | Exits the chatbot            |
